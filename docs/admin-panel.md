# Panel de administración (básico)

## Vistas principales
- **Calendario diario/semanal** con citas.
- **Filtro** por barbero y fecha.
- **Bloqueo rápido** de horarios.

## Acciones clave
- Reprogramar cita (cambio de hora/barbero).
- Cancelar cita con estado y motivo.
- Editar servicios y duración.

## Recomendaciones UI
- Tabla con estados por color.
- Confirmación antes de cambios críticos.
- Logs de modificaciones.
